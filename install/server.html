Installation du serveur non-officiel

Le tutoriel proposé ici, permet d'installer un serveur non-officiel pour le jeu Don't Play Together, sur une machine avec un système d'exploitation comportant un noyau Linux. Pour ce tutoriel nous vous recommendons Debian 10, mais tout autre système d'exploitation basé sur Debian peut également fonctionner. Vous devrez absolument avoir un accès root pour pouvoir installer le serveur. Le serveur devra comporter une connexion Internet stable. Si vous louez un serveur chez un hébergeur, il est possible que vous ne puissiez pas modifier le fichier de configuration de php. Contactez votre hébergeur pour en savoir plus.

Pour commencer, il faut installer les packages suivants :

  • apache2
  • php
  • git


Vous pouvez faire cela grâçe à la commande suivante (cette commande peut prendre du temps car le système va se mettre à jour en même temps):

1. apt-get -y update && apt-get -y upgrade && apt-get -y install apache2 php git

Accédez ensuite au dossier /var/www/html puis copier le repository DPT-Web. Pour se faire tapez les commandes suivantes :

1. cd /var/www/html
2. git clone https://github.com/Deleranax/DPT-Web.git /var/www/html/

Maintenant, il faut éditer la configuration de php :
A noter que la version de php qui est utilisée dans ce tutoriel, est la version 7.3. L'emplacement du fichier de configuration ainsi que la ligne à modifier peuvent changer !

1. nano /etc/php/7.3/apache2/php.ini

Il faut changer la ligne suivante :

1. output_buffering = 4096 À REMPLACER PAR output_buffering = off

Il ne reste plus qu'à démarrer Apache 2 et le tour est joué ! :

1. systemctl enable --now apache2

Voilà, votre serveur Don't Play Together est lancé. Si vous avez le moindre problème avec l'une de ces commandes, ou que vous avez des erreurs, vous pouvez nous contacter sur discord via le lien suivant : Dwight Studio's Hub
