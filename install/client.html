Installation du client

Windows :

Pour installer le client de jeu sur windows, rendez-vous sur la page de téléchargement et téléchargez la version correspondante à votre système d'exploitation.
Exécutez le fichier et suivez les instructions d'installation. Vous pourrez alors lancer le jeu via le raccourci sur votre bureau !
Si le jeu ne se lance pas ou se ferme tout seul, veuillez nous contacter sur discord via le lien suivant : Dwight Studio's Hub

Debian :

Pour installer le client de jeu sur les distributions basées sur Debian, rendez-vous sur la page de téléchargement et téléchargez la version correspondante à votre système d'exploitation.

Ouvrez un terminal à l'emplacement du fichier et tapez la commande suivante :

1. sudo dpkg -i dpt_1.0.0-BETA-Linux.deb


Vous pourrez lancer le jeu en cliquant sur le raccourci dans le dock.
Si le jeux ne se lance pas ou se ferme tout seul, veuillez nous contacter sur discord via le lien suivant : Dwight Studio's Hub

Pour les utilisateurs de l'environement GNOME, afin d'obtenir une immersion optimal, il est nécessaire de cacher le dock. Pour cela, dans les paramètres, il y a une option "Masquer automatiquement le dock". Il est également conseiller de cacher la barre en haut de l'écran. Pour ce faire rendez-vous sur Gnome.org et installez l'extension "Intégration GNOME Shell" pour votre navigateur et suivez la procédure afin d'installer le connecteur d'extension. une fois fait, il suffit de rafraîchir la page Gnome.org et de cliquer sur le boutton Off pour activer l'extension. La barre devrait alors se cacher.
